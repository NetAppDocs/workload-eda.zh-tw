= 將 EDA 與 Perforce 集成
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
將 EDA 與 Perforce Helix Visual Client (P4V) 集成，以便開發人員可以使用 Perforce CLI 管理您的工作區。這使得開發人員能夠快速地在專案和工作區之間切換，從而節省開發時間。

.步驟
. 下載 https://builders.console.workloads.netapp.com/p4v["P4V整合文件"^]。
. 開啟 P4V，然後前往「工具」>「管理工具」>「HTML 標籤」。
. 選擇“導入 HTML 選項卡”。
. 選擇 P4V 整合 XML 文件，然後選擇「導入」。
. 轉到“視圖”>“工作負載工廠”。


.結果
NetApp Workload Factory for EDA Web UI 在 P4V 用戶端中顯示為 HTML 標籤。
